## What this does

Queries five molecular databases to identify and score candidate targets:

| Layer | Database | What it measures |
|-------|----------|-----------------|
| L1 | GWAS Catalog | Shared genetic architecture |
| L2 | Open Targets Platform | Target–disease association scores |
| L3 | gEAR / HPA | Inner ear expression |
| L4 | STRING | Protein interaction networks |
| L5 | Literature | Experimental / clinical evidence |

Final output: ranked candidate list with Tier 1/2/3 classification,
exported to Excel with PubMed search strings for each target.
